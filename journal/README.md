# Journal

The `journaler` agent writes one dated file per trading day here
(`YYYY-MM-DD.md`): every thesis, conviction score, decision, your approval/rejection,
and any orders actually placed.

Review this weekly. The honest question it should answer over time: **is the system
actually beating a simple "buy and hold the S&P 500" — on a risk-adjusted basis,
after costs?** If after a fair stretch it isn't, that's the most important signal the
journal can give you.
